# WBParser

---

Парсер карточек товаров с Wildberries: собирает товары по запросу, фильтрует по цене, рейтингу и стране производства, затем сохраняет результат в Excel.

## Основные возможности

---

- Сбор товаров по поисковому запросу  
- Фильтр по цене (до 10 000 ₽) и стране производства (Россия)  
- Фильтр по рейтингу ≥ 4.5 и количество отзывов  
- Сохранение результатов в `wildberries_palto_filtered.xlsx`

## Установка и запуск

---

1. Убедитесь, что у вас установлен Python ≥ 3.8.
2. Проект автоматически:
   - Установит зависимости из `requirements.txt`  
   - Установит браузер Chromium для Playwright  

3. Запустите парсер:
   ```bash
   python main.py
   ```

После завершения работы в папке появится файл `wildberries_palto_filtered.xlsx` с отфильтрованными товарами.
